# Icons for PWA

To generate icons, open `generate-icons.html` in your browser and click "Generate All Icons". 
Then click the download buttons to save each icon to this directory.

Required icon files:
- icon-72x72.png
- icon-96x96.png
- icon-128x128.png
- icon-144x144.png
- icon-152x152.png
- icon-192x192.png
- icon-384x384.png
- icon-512x512.png
